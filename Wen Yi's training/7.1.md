`bool`值：逻辑值，只有True(真)或者 False(假)
逻辑比较：
`==` 等于 也可以写成`is` (注意一个等于号永远是赋值)
`> < = >= <=` 字面意思
`!=` 不等于 也可以写成 `is not`
逻辑运算：
`and` 与 (两个都是 True 才返回 True)
`or` 或 (只要有一个 True 就返回 True)
`not` 非 (一元运算符，返回相反 bool 值)
`^` 异或 (只有两个量一个是 True 一个是 False 才返回 True)

class: 类，类型
e.g. 
```python
class example:
	def __init__(self, a, b):
		self.a = a
		self.b = b
		"""
		self 表示这个类自己，只要是类里面的成员函数第一个变量都必须是self
		__init__()是类的构造函数，每创建一个类的
		这里的self.a,self.b是成员变量，是example类里拥有的变量，而a, b是传进来的参数
		"""
		
ex = example(1, 2) # example是类型，ex是这个类型的一个实例/对象 (instance / object)
"""
实际上调用了example.__init__(self, 1, 2)
ex.a = 1
ex.b = 2
"""
```
在如上代码块中，有着超过一行的注释，请用三个单引号/双引号 ("""aaa""" / '''aaa''')括起来
